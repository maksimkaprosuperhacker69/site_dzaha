Сайт написан на языке Python, на фреймворке Flask.

(К каждому блоку кода есть коментарии)


client_secrets.json:

Этот файл содержит клиентские секреты, которые используются для
аутентификации на стороне клиента, например, при работе с API Google Cloud.

**** Для того что бы файлы загруженные на сайт поподали на ваш google Drive. Перед использованием проекта необходимо 
заменить  файл client_secrets.jon на свой собственный(преименовать на такое же название),который вы получите после создания OAuth токена на сайте Google Cloud Platform.
при этом очистите файл credentials.json и запишите данные из своего client_secrets.json в settings.yaml(указано в файле)
,при локальном запуске с вашим файлом client_secret.jsonфайл credentials.json сам перезапишеться ****


Для получения этого файла следует создать проект на Google Cloud Platform, настроить
авторизацию OAuth и сгенерировать JSON-файл с клиентскими секретами, который будет использоваться в вашем проекте.

Важно хранить этот файл в безопасном месте и не делиться им с посторонними лицами, чтобы обеспечить безопасность вашего проекта.

Видео гайд: https://www.youtube.com/watch?v=QxVl8m54vnk


1.app.py:

Этот файл является главным скриптом приложения Flask.
Для запуска сайта необходимо выполнить этот файл.
Приложение будет доступно по адресу localhost:5000 (или другому порту, если указано в коде).

2.Папка templates:

В этой папке содержатся HTML-шаблоны, которые отображаются в браузере пользователя.
Каждый HTML-файл соответствует определенной странице сайта или разделу.

3.4.Папка static/css:

Здесь хранятся файлы таблиц стилей (CSS), которые определяют внешний вид и форматирование элементов на веб-страницах.
CSS-файлы используются для стилизации различных компонентов сайта.

Папка static/js:

В этой папке содержатся файлы JavaScript, которые могут содержать скрипты, обрабатывающие взаимодействие пользователя с веб-страницей.
JavaScript-файлы могут быть использованы для реализации интерактивных функций и обработки событий на стороне клиента.

4.Папка static/images:

Здесь хранятся изображения и другие медиа-файлы, которые используются на веб-страницах.
Эти файлы могут быть загружены и отображены на страницах сайта, чтобы предоставить визуальные элементы для пользователей.
